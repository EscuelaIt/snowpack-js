## Requirements

1. Install [Node 12.18.2](https://nodejs.org/dist/v12.18.2/)
2. Clone project
3. `cd snowpack-js`
4. Install dependencies: `npm i`
5. Remember to install also dependencies of the subprojects:

    ```
    cd snowpack-build-scripts
    npm i
    ```
